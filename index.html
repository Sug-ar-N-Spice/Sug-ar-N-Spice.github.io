Projects

Project 1: K-means Algorithm and PCA

K-means Algorithm and Principal Component Analysis (PCA) - Used CryptoCurrency Data to apply principles of K-means algorithm and principal component analysis (PCA) to classify cryptocurrencies according to their price fluctuations across various timeframes. This method specifically examines price changes over intervals spanning 24 hours, 7 days, 30 days, 60 days, 200 days, and 1 year.

Project 2: Incident Ticket Prediction Model

Incident Ticket Prediction Model - A scalable AI-powered solution designed to predict ticket turn-around-times and assist with resource allocation, phasing, and structuring for high-performance teams. G4 version 1 (G4v1) is a classification model achieving 70% accuracy. This approach uses AI and machine learning to streamline incident management.

  • Analysis of medical datasets, including STI-related data and scientific literature.
  • Integration with pre-trained AI models for enhanced conversational experiences.
